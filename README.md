# 🧠 Brain Tumor Detection using Ensemble Deep Learning

## 📌 Overview

This project focuses on automatic brain tumor detection from MRI images using deep learning techniques. The research aims to improve detection performance by combining multiple CNN models with ensemble learning, explainable AI, uncertainty estimation, and cross-dataset evaluation.

This project is being developed as a Final Year Research Project.

---

## 🎯 Objectives

- Detect brain tumors from MRI images.
- Train and compare EfficientNet and DenseNet121 models.
- Develop an ensemble learning framework.
- Improve model interpretability using Grad-CAM.
- Estimate prediction uncertainty.
- Evaluate cross-dataset generalization.

---

## 📂 Datasets

### Primary Dataset
- BRISC 2025 (Brain Tumor Segmentation and Classification)

### Secondary Dataset
- Brain Tumor MRI Dataset (Masoud Nickparvar)
